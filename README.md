# Guru Jambheshwar Mandir Parta - Full Stack Website

A comprehensive full-stack website for Guru Jambheshwar Mandir Parta with public pages and a secure admin panel.

## Features

### Public Website
- **Home Page**: Welcome page with temple background image
- **About Page**: Temple history and information (editable by admin)
- **Location Page**: Embedded Google Map and location details
- **Videos Page**: YouTube channel integration with embedded videos

### Admin Panel
- **Secure Login**: Session-based authentication
- **Dashboard**: Quick access to all admin functions
- **Content Management**: Edit home, about, and location page content
- **Image Upload**: Change background images
- **Video Management**: Add/remove YouTube video links
- **Database Integration**: All changes stored in MySQL
